# 2026-06-24 회원/지도 UI 변경 요약

## 변경 내용

- 회원 관리 화면을 별도 페이지로 정리했다.
- 내 정보는 기본적으로 이메일, 이름, 전화번호를 텍스트로만 보여준다.
- `수정` 버튼을 눌렀을 때만 이름과 전화번호 입력 필드를 열고, 저장 또는 취소 시 다시 읽기 전용 상태로 돌아간다.
- 회원 검색은 관리자 계정에만 상단 메뉴로 노출한다.
- 지도 로드 실패 시 오른쪽 지도 영역에 원인을 안내하는 메시지를 표시한다.

## 로컬 설정

- Kakao 지도는 프론트 `.env`의 `VITE_KAKAO_MAP_API_KEY` 값을 사용한다.
- 이 값은 Kakao Developers의 JavaScript 키여야 한다.
- Kakao Developers Web 플랫폼에는 `http://localhost:5173` 또는 실제 접속 주소를 등록해야 한다.

## 확인한 검증

- `npm run build`
